D - Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement.

  • si vos éléments sont organisés horizontalement : l'axe principale (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe vertical.
  • si vos éléments sont organisés verticalement : l'axe principale (main axis) est l'axe verticale, l'axe secondaire (cross axis) est l'axe horizontale.

1 - Alignement sur l'axe principale

Afin d'aligner nos éléments sur l'axe principal, on utilise la propriété du flex Justify-content

Cette propriété gère la distribution des emplacements des flex-items suivant la place disponible dans le flex-conteiner. Elle le fait, selon le positionnement de l'axe principal

  • flex-start : aligner au début (valeur par défaut)
  • flex-end : aligner à la fin
  • flex-center : aligner au centre
  • space-between : les éléments sont étirés sur tout l'axe (il y a de l'espace entre eux)
  • space-around : les éléments sont étirés sur tout l'axe, mais ils laissent aussi de l'espace sur les extrémités
  • space-evenly : les éléments sont répartient uniformément dans le conteneur d'alignement le long de l'axe principal

E - Retour à la ligne

Par défaut les blocs essaient de rester sur la même ligne même s'ils n'ont pas la place (ce qui provoque parfois des problèmes de conception). Si vous le souhaitez, vous pouvez demlmander au bloc de passer à la ligne lorsqu'ils n'ont plus de place en utilisant la propriété flex-wrap

Elle peut prendre les valeurs suivantes :

  • nowrap : pas de retour à la ligne (la valeur par défaut)
  • wrap : les éléments vont à la ligne lorsqu'il n'y a plus de place
  • wrap-reverse : les éléments vont à la ligne lorsqu'il n'y a pas de place en sens inverse

F - Répartir plusieurs lignes

Si vous avez plusieurs ligne dans votre flexbox,vous pouvez choisir comment celles-ci seront réparties avec align-content. Cette propriété permet de gérer l'alignement des différents lignes (ou colonnes) les un par rapports au autres. Cette propriété n'aura d'éffet que si le conteneur flexible posséde plusieurs lignes(si l'axe principale est horizontal) ou plusieurs colonnes (si l'axe principle est vertical) et que si le flex-wrap est présent.

Les valeurs de cette propriété :

  • flex-start : les éléments sont placés au début
  • flex-end : les élélements sont placés à la fin
  • center : les éléments sont placés au centre
  • space-between : les éléments sont séparés avec de l'éspace entre eux
  • space-around : les éléments sont séparés avec de l'éspace entre eux, mais il y'a aussi de l'éspace au début et à la fin
  • space-evenly : Les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal.
  • stretch (par défaut) : les éléments s'étirent pour occuper tout l'éspace
